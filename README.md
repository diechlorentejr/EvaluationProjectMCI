This git incluedes the code from our MCI group Project. The project was created in a private git because we dont want to train Microsoft AI with our code.
Final changes:

Evaluation Reward System Added:

    Students now receive a bonus for each completed evaluation (default: 1% per evaluation), up to a configurable maximum bonus (e.g. 3%)

Progress Tracker on “My Evaluations”, A new progress tracker shows, per course:

    Number of completed evaluations
    Number of remaining evaluations
    Total bonus earned so far

Lecturer Settings During Course Creation Lecturers can now define:

    How many evaluations are required for a course
    How much bonus each evaluation grants

Participant Count Display for Lecturers:

    The lecturer dashboard now shows the total number of students who participated in the course sessions

UI Improvements:

    Input fields for course name, evaluation target, and bonus value are now uniformly sized and properly aligned

Stylesheet Link Fixed:

    The main CSS file is now correctly linked, ensuring consistent and reliable UI styling
